**3A Joomla** is a set of utilities designed to simplify certain aspects of working with the Joomla CMS. It is a plug-in for the [Pyrog](https://ironmesh.ru/ru/products/22-prilozheniya/71-pyrog) software suite.


It includes the following modules:

- **Sigplus filler** — a utility for convenient and fast creation of data files used by the [Sigplus](https://extensions.joomla.org/extension/sigplus/) plugin to assign descriptions to images in a gallery.
