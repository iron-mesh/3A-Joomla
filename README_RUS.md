**3A Joomla** — это набор утилит, который призван облегчить некоторые аспекты работы с CMS Joomla. Является подключаемым плагином для программного комплекса [Pyrog](https://ironmesh.ru/ru/products/22-prilozheniya/71-pyrog).

Руководство по установке и использованию читайте в [материале](https://wiki.ironmesh.ru/products:pyrog-plugins:3a-joomla:main).

Включает в себя следующие модули:

- **Sigplus filler** — утилита для удобного и быстрого создания файлов с данными, которые используются плагином [Sigplus](https://extensions.joomla.org/extension/sigplus/) для присваивания описаний изображениям в галерее.
